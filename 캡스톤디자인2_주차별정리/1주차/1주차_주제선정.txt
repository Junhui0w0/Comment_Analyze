[1주차 - 주제선정 // AI & 빅데이터 분석]
1. 유튜브 댓글 감정 분석
	-정책 또는 사건에 대한 동영상에 달린 댓글을 분석
	-트렌드 분석(주식) (2차 분석 - 어떤 키워드(이유) 때문에 긍정으로 판별했는지?)
		-> 1. 특정 기업에 관한 뉴스(사건관련 영상) 등장 
		-> 2. 영상의 댓글 분석해서 댓글 여론이 성장, 쇠퇴인지 판별
		-> 3. 성장,쇠퇴의 이유 키워드
	-가중치
		-> 1. 댓글 좋아요 / 대댓글 수
		-> 2. 추출된 핵심 키웓 수
	-이외에도 어느 부분은 어떻게 개선하면 좋겠다. 라는 내용도 요약 및 출력
	-다만, 비꼬는 댓글도 분석이 가능한지 ?
		-> 미세 조정(Fine tuning)을 통해 기존 모델 + 새로운 데이터 추가 학습
		-> GPT API로 통합해서 안정적으로 제공 (개추★)
	-기술적으로 부족할 수 있는데.. 추가 기술 ?
		-> 
	-댓글 알바(여론조작) 어떻게 막을건지 ?
		-> 1. 계정 생성 기간 (인정)
		-> 2. 댓글은 단 채널들의 방향성 판별 (비추)
			(ex: 95% 이상이 정치 관련이면 의심해볼 수 있음)

	-리포트 툴

	1-1) 데이터 수집: Google API를 통해 선택한 동영상들의 댓글 수집 가능
	1-2) 수집 제한: 기본 API로는 하루 추출 댓글 수가 부족할 수 있음
	1-3) 해결방안: Google에 따로 신청하면 제한 해결 가능
	
	2-1) 댓글 감정 분석: Kobert, Ko-Bert, DistilBERT 등 각 모델 성능 비교
	2-2) GPT 활용: GPT API를 통해 추출한 댓글들의 감정 분석
	2-3) 최종분석: 기존 분석 내용과 GPT 분석의 평균값으로 감정 판별

	3-1) 데이터 시각화: matplotlib 또는 Seaborn을 이용해 감정 분석 결과 시각화
